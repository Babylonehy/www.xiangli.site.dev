## Zotero using SCI-hub

1. Find ```extensions.zotero.findPDFs.resolvers``` in ```preferences -> advanced -> config editor```.

2. Change the vaule to
```
{
    "name":"Sci-Hub",
    "method":"GET",
    "url":"https://sci-hub.st/{doi}",
    "mode":"html",
    "selector":"#pdf",
    "attribute":"src",
    "automatic":true
}
```
